# Python Fundamentals

A beginner-friendly Python project demonstrating core programming concepts through a simple student profile script.

## What It Does

`student_profile.py` stores a student's information, displays it in a formatted profile card, and checks placement eligibility based on CGPA.

**Sample Output:**
```
==============================
       STUDENT PROFILE
==============================
Name       : Krishna Chaitanya Reddy Kothakapu
Age        : 22
City       : Mahabubnagar
University : University of Pittsburgh
CGPA       : 7.15
==============================
Eligible: CGPA 7.15 is above 7.0
```

## Concepts Covered

- **Variables** — storing strings, integers, and floats
- **Functions** — defining and calling reusable blocks of code
- **f-strings** — formatted string output
- **Conditional statements** — `if/else` logic for eligibility check
- **String repetition** — using `*` to build separators

## How to Run

Make sure you have Python 3 installed, then run:

```bash
python student_profile.py
```
